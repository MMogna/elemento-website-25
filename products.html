Four Ways to Deploy Your Cloud

Choose the deployment model that fits your infrastructure needs and preferences

1. Private AtomOS

Deploy AtomOS on your own private servers. Full control over hardware, networking, and security with autonomous clustering via C4 protocol. Learn more about our C4 protocol technology and deployment support.

Self-Hosted Full Control

2. Colocation AtomOS

Deploy AtomOS on bare-metal cloud servers from providers like Hetzner, OVH, or Scaleway. Enterprise-grade virtualization with dedicated hardware.

Bare Metal Dedicated

3. Shared AtomOS

Use VMs from Elemento's shared AtomOS infrastructure. Managed by Elemento with enterprise features and support included.

Managed Shared

4. Native CSP VMs

Deploy native VMs (EC2-like) directly on cloud providers through Atmosphere. Access AWS, Azure, GCP with unified management.

Native Multi-Cloud
